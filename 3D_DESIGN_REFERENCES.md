# Inspiración para Diseños 3D - Referencias Web 2024-2025

## Sitios Premium con Efectos 3D Elegantes

### 1. **Active Theory** (Recomendado)
- **URL:** https://activetheory.net/
- **Características:** Entorno 3D gamificado, controlable con mouse/scroll/drag
- **Efectos:** Perspectiva interactiva, animaciones fluidas, parallax avanzado
- **Aplicable a:** Hero section, portfolio interactivo

### 2. **Chirpley**
- **Características:** Visualización 3D de soluciones IA
- **Efectos:** Cards con rotación 3D, transiciones suaves
- **Aplicable a:** Servicios, testimonios

### 3. **Peter Tarka Portfolio**
- **URL:** https://www.petertarka.com/
- **Características:** Portfolio con visualización 3D cautivadora
- **Efectos:** Dynamnic animations, color vibrante, scroll interactivo
- **Aplicable a:** Proyectos, experiencias

### 4. **Deed Delivery**
- **Características:** Animaciones de entrada dinámicas
- **Efectos:** Scroll interactivo, transiciones light-to-dark, efectos de scroll
- **Aplicable a:** Secciones, transiciones entre componentes

### 5. **Locomotive JS**
- **URL:** https://locomotivemtl.com/
- **Características:** Scroll fluido con efectos cursor-reactivos
- **Efectos:** Typography dinámica, parallax visual, scroll suave
- **Aplicable a:** Experiencia general, scroll stories

## Técnicas 3D Implementables

### Card Hover 3D (CSS + React)
\`\`\`css
transform: perspective(1200px) rotateX(-5deg) rotateY(5deg) translateZ(20px);
transition: transform 0.3s ease;
\`\`\`

### Parallax Mouse Movement
Detectar posición del mouse y aplicar transformaciones sutiles a capas diferentes.

### Scroll-Triggered Animations
Usar intersection observer para animar elementos al hacer scroll.

### Gradient Animations
Gradientes que rotan continuamente con efecto neon.

## Libraries Recomendadas

- **Framer Motion:** Animaciones 3D declarativas
- **Three.js:** Gráficos 3D avanzados
- **GSAP:** Timeline animations profesionales
- **Lenis:** Scroll smooth de lujo
- **Locomotive Scroll:** Scroll interactivo
